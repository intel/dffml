GSoC 2020¶

DFFML participated in Google Summer of Code under the Python Software Foundation umbrella. You can read all about what this means at https://python-gsoc.org/

Student Contributions¶

Huge thanks to our students of the 2020 GSoC program who made a huge impact on DFFML’s abilities, collection of models, and tutorials. As well as various bug fixes.

Himanshu @0dust¶

Project: NLP (Natural Language Processing) Models and Operations

  • Classification model based on Transformers architecture

  • Question Answering model based on Transformers architecture

  • Spacy model for Named Entity Recognition

  • Operations to wrap Spacy functions for text preprocessing

  • Operations to wrap Scikit-learn functions for text preprocessing

Saksham @sakshamarora1¶

Project: Integrating Image Processing into DFFML

  • Wrapping important image pre-processing functions as DFFML operations

  • PyTorch based pre-trained Convolutional Networks for image classification

  • Custom Neural Networks for image pre-processing and classification tasks

Aghin @aghinsa¶

Project: Improving DataFlows and Distributed Orchestrator

  • DataFlow Locking Tutorial

  • DataFlow Continuous Deployment Tutorial

  • Distributed Orchestrator based on NATs

Thanks¶

Big thanks to Terri her work organizing GSoC and letting us copy her format she used for CVE Binary Tool, another awesome project with a security focus that was a part of GSoC 2020 as well. Check them out too!
