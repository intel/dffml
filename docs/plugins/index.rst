Plugins
=======

DFFML is plugin based. The following is a list of plugins which provide
implementations for the various abstractions defined by DFFML.

If you have developed a plugin for DFFML and would like it listed here, please
open an `issue <https://github.com/intel/dffml/issues/new?assignees=&labels=documentation&template=new_plugin.md&title=plugin%3A+new%3A+>`_.
