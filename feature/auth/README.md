## Usage

$ dffml operations repo \
  -sources memory=nill \
  -log debug \
  -keys feedface \
  -repo-def UnhashedPassword \
  -ops scrypt get_single \
  -opimpn-memory-opimps scrypt get_single \
  -remap get_single.ScryptPassword=scrypt \
  -output-specs '["ScryptPassword"]=get_single_spec'
