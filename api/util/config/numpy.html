Util Config Numpy¶

This file handles converting callables with numpy docstrings into config classes by parsing their docstrings to find their default values, finding the help text for each value, and then calling make_config to create a config class representing the arguments to that callable.

dffml.util.config.numpy.make_config_numpy(name: str, cls: Type, properties: Optional[Dict[str, Tuple[Type, dffml.base.field]]] = None)[source]¶

Given a numpy class, read its docstring and __init__ parameters to generate a config class with properties containing the correct types, and default values.
